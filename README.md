# Customer Segmentation for Retail Using Machine Learning

This project focuses on segmenting retail customers based on their purchasing behavior using machine learning techniques.

## Project Objective
To identify distinct customer segments using transaction data and analyze customer behavior using RFM (Recency, Frequency, Monetary) analysis.

## Methodology
- Data Cleaning and Preprocessing
- Feature Engineering using RFM analysis
- Feature Scaling
- Customer Segmentation using Clustering
- Cluster Interpretation and Business Insights
- Visualization of Customer Segments

## Tools and Technologies
- Python
- Pandas, NumPy
- Matplotlib
- Machine Learning concepts (Clustering)

## Key Outcomes
- Identification of loyal, at-risk, occasional, and inactive customers
- Actionable business insights for targeted marketing strategies
